This is my first ever python project. 
It is a time capsule - a memory of a more innocent time.
The time when one was not constrained by social constructs such as OOP or FP. When the only honest and true way was procedural programming done in a single method that did everything; the ancient and majestic ``doEverything()`` method.
It reminds us of a simpler time - a time when your biggest problem were icompetent and WoW-addicted students working with you on a group project. 
It was a time before AI and vibe coding. Such code as this is impossible to replicate today. Not a single coding agent is capable of creating this code that is functional yet terrible.

For these memories it is of paramount importance that we preserve this piece of software history forevermore like an ancient mosquito incased in amber.
